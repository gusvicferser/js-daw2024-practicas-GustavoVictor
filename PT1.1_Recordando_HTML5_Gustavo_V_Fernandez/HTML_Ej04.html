Según la Wikipedia:

  • URL (Uniform Resource Locator): Es una dirección única que identifica un recurso en la web, como una página web o un archivo. Consiste en un protocolo (como HTTP o HTTPS), un nombre de dominio (como www.example.com), y una ruta que puede incluir un archivo específico​
  • HTTP (Hypertext Transfer Protocol): Es el protocolo de comunicación utilizado para transferir información en la web. Sigue un esquema de petición-respuesta entre un cliente (como un navegador) y un servidor. HTTP es sin estado, lo que significa que no guarda información sobre peticiones anteriores​
  • TCP/IP (Transmission Control Protocol / Internet Protocol): Es un conjunto de protocolos que forman la base de Internet. TCP se encarga de asegurar la entrega confiable de datos entre dos puntos, mientras que IP se ocupa de direccionar y enrutar los paquetes de datos para que lleguen al destino adecuado
  • Internet: Es una red global de redes interconectadas que permite la comunicación y el intercambio de información entre millones de dispositivos en todo el mundo. Utiliza el conjunto de protocolos TCP/IP para funcionar y se basa en principios como el de extremo a extremo y la robustez​
  • W3C (World Wide Web Consortium): Es la organización internacional encargada de desarrollar estándares para la web, como HTML y otros lenguajes de marcado. Fundada por Tim Berners-Lee, el W3C busca asegurar que la web sea accesible para todos​
  • HTML (HyperText Markup Language) Es el lenguaje de marcado utilizado para estructurar el contenido de las páginas web. Mediante etiquetas, permite definir elementos como párrafos, enlaces, imágenes y tablas, los cuales son interpretados por los navegadores para mostrar el contenido de manera adecuada​
